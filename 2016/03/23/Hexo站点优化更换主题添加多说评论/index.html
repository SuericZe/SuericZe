前三篇的博客中我们介绍了Hexo的站点配置及与GitHub的关联操作,这篇主要是介绍Hexo博客的主题更换及添加多说评论,至于其他功能,就有待小伙伴自己去开发了~~

更换主题

目前大家用的比较好看的主题主要有 yilia 和 pacman ,两种样式分别如下:

主题使用步骤:

安装(在任意文件夹右键,选择 Git Bash,执行命令):

$ git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

修改F:\Hexo下的 _config.yml配置文件,更改thme标签的值为yilia, 注意冒号后面的空格

然后进入F:\Hexo\themes\yilia执行git pull 命令即可将主题同步到博客上

最后,在F:\Hexo下右键,选择Git Bash执行Hexo g 和 Hexo d命令后,输入GitHub用户名密码后既可查看效果

同理,安装pacman主题也是同样的操作,执行的Git命令为

$ git clone https://github.com/A-limon/pacman.git themes/pacman

小贴士:

  • yilia的github地址为:https://github.com/litten/hexo-theme-yilia
  • pacman的github地址为: https://github.com/A-limon/pacman
  • 附上主题集合贴:点我

注意:如果想修改默认的展示效果,可以进入F:\Hexo\thmes\pacman中更改_config.yml 的主题配置文件进行设置,文件内有注释表明每个模块的作用,更改后记得在该文件夹下执行git pull命令

添加多说功能

因为Hexo默认使用国外比较流行的dispus,但因地制宜,我们一般使用博客就会改为多说评论,支持国产,而且使用多说超级简单,只需要一分钟不到的时间就可以让你的博客实现评论功能.

首先,去多说使用社交账号登陆,因为多说不支持注册,所以登陆后会让你设置一些站点信息,如图所示

大家注意:在域名设置那里,前面的即为自己独一无二的多说short name, 记住该前缀后,如果使用的是yilia 或者 pacman 主题的话,只需要前往F:\Hexo\thmes\pacman下,打开_config.yml,在duoshuo模块下,将前缀修改即可,是不是so easy;

注意:附上官方教程,适用于使用了light主题

添加百度统计

为了更好的知道并了解我们的网站被多少人访问了,推荐使用百度统计,当然谷歌统计也是可以的,但天朝的国度你可能需要搭梯子…

  1. 注册并登陆百度统计
  2. 在网站中心–>代码获取中点击复制代码
  3. 打开F:\Hexo\themes\pacman\layout\after_footer.ejs文件,把刚才复制的代码复制到最下面即可;
  4. 然后在网站中心–>代码安装检查中检测代码是否添加成功
